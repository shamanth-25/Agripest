🌱 AgriPest
An AI-powered smart agriculture project focused on pest detection and crop protection using Machine Learning and Deep Learning techniques.
AgriPest helps farmers identify harmful pests early and supports better agricultural decision-making.
🚀 Features
🔍 Pest Detection using AI/ML
📷 Image-based crop pest identification
🤖 Deep Learning powered predictions
🌾 Supports smart farming practices
📊 Easy-to-use interface for predictions
⚡ Fast and accurate detection system
🛠️ Technologies Used
Python
Machine Learning
Deep Learning
TensorFlow / Keras
OpenCV
NumPy & Pandas
Flask / Streamlit (if used)

⚙️ Installation
Clone the repository:
Bash
git clone https://github.com/shamanth-25/Agripest.git
Move into the project folder:
Bash
cd Agripest
Install dependencies:
Bash
pip install -r requirements.txt
Run the project:
Bash
python app.py
